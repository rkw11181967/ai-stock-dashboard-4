# 📊 AI Stock Signal Dashboard (News + AI + Technical Analysis)

A cloud-based AI-powered stock signal engine that scans financial news, analyzes sentiment across multiple headlines, applies technical confirmation, and generates ranked trading signals.

---

## 🚀 Features

### 🧠 AI News Analysis
- Scans market news via RSS feeds
- Extracts stock tickers automatically
- Uses AI to evaluate:
  - Sentiment (Bullish / Bearish / Neutral)
  - Confidence (0–1)

---

### 📊 Multi-Headline Scoring (Advanced)
- Aggregates multiple headlines per stock
- Averages sentiment confidence
- Reduces noise and false signals

---

### 🎯 Signal Engine

#### 🚀 Strong Signals
- BUY → Confidence ≥ +0.85 + technical confirmation
- SELL → Confidence ≥ -0.85 + technical confirmation

#### 👀 Watch Signals
- WATCH (LEAN BUY) → ≥ +0.75
- WATCH (LEAN SELL) → ≤ -0.75

---

### ⚙️ Technical Analysis
- 5-day vs 20-day moving average crossover
- Confirms trend direction

---

### 🏆 Ranking System
- Ranks stocks by signal strength
- Outputs **Top 5 opportunities only**

---

### 🔔 Alerts
- 📧 Email alerts for strong BUY/SELL signals
- 👀 Watch signals shown on dashboard only

---
